## 📌 Overview

The Smart Gym System is designed to help gyms manage their daily activities digitally instead of using manual methods. It provides an easy-to-use interface for handling members, trainers, and booking sessions.

---

## 🚀 Features

- 👤 **Member management** (add, view, update, delete)  
- 🏋️ **Trainer management**  
- 📅 **Reservation system**  
- 🔄 **Real-time interaction between frontend and backend**  
- 📊 **Organized database storage**  

---

## 🛠️ Technologies Used

### 🎨 Frontend
- React.js

### ⚙️ Backend

- Express.js  

### 🗄️ Database

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash id="u5m1vx"
git clone https://github.com/your-username/smart-gym-system.git
cd smart-gym-system
```

---

### 2️⃣ Setup Backend

```bash id="q9r4ld"
cd smart-gym-backend
npm install
npm start
```

---

### 3️⃣ Setup Frontend

```bash id="p3k8az"
cd smart-gym-frontend
npm install
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file in the backend folder and add:

```env id="x6t2mf"
MONGO_URI=your_mongodb_connection_string
PORT=5000
```
